# こうかとんモンスターバトル

## 実行環境の必要条件

- python >= 3.10
- pygame >= 2.1

## ゲームの概要

- こうかとん風のモンスターを操作して，敵モンスターと1対1で戦うターン制バトルゲーム
- ポケモンのようにコマンドを選択して戦う
- 敵のHPを0にするとゲームクリア
- 自分のHPが0になるとゲームオーバー
- バトル中に1回だけメガシンカを使うことができる

## ゲームの遊び方

- 上下キーでコマンドを選択する
- Enterキーでコマンドを決定する
- 「たたかう」を選ぶと敵にダメージを与える
- 「まもる」を選ぶと次の敵の攻撃ダメージを軽減する
- 「メガシンカ」を選ぶと，1回だけプレイヤーの攻撃力が上がる
- 敵はプレイヤーの行動後に自動で行動する
- 敵のHPが0になるとゲームクリア
- 自分のHPが0になるとゲームオーバー

## ゲームの実装

### 共通基本機能

- 背景，プレイヤーモンスター，敵モンスターの描画
- プレイヤーと敵のHP表示
- コマンド選択画面の表示
- 上下キーによるコマンド選択
- Enterキーによるコマンド決定
- プレイヤーターンと敵ターンの切り替え
- 攻撃処理
- 勝利画面，ゲームオーバー画面

### 分担追加機能

- メガ進化機能
- 技追加
- 敵のランダム行動
- タイトル画面，ゲームオーバー画面
- 音声、効果音

## 追加機能：味方技追加

### 担当内容

- 味方が使用できる技を追加する
- 「まもる」の仕様を本家ポケモンに近い形へ変更する

### 追加する技

- おにび
  - 命中率100%
  - 敵をやけど状態にする
  - やけど状態の敵は毎ターン固定ダメージを受ける

- はねやすめ
  - 自分の最大HPの50%を回復する
  - 最大HPを超えて回復しない

- ブレイズキック
  - 命中率100%
  - 敵に通常攻撃より大きいダメージを与える

### まもる機能の変更

- 「まもる」を使用すると，敵の攻撃をすべて防ぐ
- 連続で使用していない場合は必ず成功する
- 連続で使用した場合，成功率は1/3になる
- 「まもる」以外の技を使用すると，連続使用扱いをリセットする
