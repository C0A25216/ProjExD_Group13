# こうかとんモンスターバトル

## 実行環境の必要条件

- python >= 3.10
- pygame >= 2.1

## ゲームの概要

- こうかとん風のモンスターを操作して，敵モンスターと1対1で戦うターン制バトルゲーム
- ポケモンのようにコマンドを選択して戦う
- 敵のHPを0にするとゲームクリア
- 自分のHPが0になるとゲームオーバー
- バトル中に1回だけメガシンカを使うことができる

## ゲームの遊び方

- 上下キーでコマンドを選択する
- Enterキーでコマンドを決定する
- 「たたかう」を選ぶと敵にダメージを与える
- 「まもる」を選ぶと次の敵の攻撃ダメージを軽減する
- 「メガシンカ」を選ぶと、1回だけメガシンカし、攻撃力が上昇するとともに受けるダメージが軽減される
- 敵はプレイヤーの行動後に自動で行動する
- 敵のHPが0になるとゲームクリア
- 自分のHPが0になるとゲームオーバー

## ゲームの実装

### 共通基本機能

- 背景，プレイヤーモンスター，敵モンスターの描画
- プレイヤーと敵のHP表示
- コマンド選択画面の表示
- 上下キーによるコマンド選択
- Enterキーによるコマンド決定
- プレイヤーターンと敵ターンの切り替え
- 攻撃処理
- 勝利画面，ゲームオーバー画面

### 分担追加機能

- メガ進化機能
- コマンド「メガシンカ」を追加
  - メガシンカ演出（mega.jpg → megakoukaton.png）
  - メガシンカ後は攻撃力を上昇
  - メガシンカ後は被ダメージを30%軽減
  - メガシンカはゲーム中1回のみ使用可能
  - 使用後はコマンドを灰色表示し、再選択できないように変更
- 技追加
- 敵のランダム行動
- タイトル画面，ゲームオーバー画面
- 音声、効果音
